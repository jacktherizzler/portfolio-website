Predicting Hand Written Digits Using Neural Networks

Python, Numpy, Pandas, Machine Learning

MNIST (Modified National Institute of Standards and Technology) dataset samples include handwritten digits totaling 70,000 images, with 60,000 examples in the training set and 10,000 examples in the testing set, both with labelled images ranging from 10 digits (0 to 9). This is a small segment from the NIST wide set where the size was normalised to fit a 20*20 pixel box without changing the aspect ratio. Handwritten digits are images in the form of 28*28 grey scale intensities of images representing an image, with the first column for each image being a label (0 to 9). The same has been chosen for the case of the testing set as 10,000 images with labels ranging from 0 to 9.

Linear Regression is a Supervised Machine Learning Model for finding the relationship between independent variables and dependent variable. Linear regression performs the task to predict the response (dependent) variable value (y) based on a given (independent) explanatory variable (x). So, this regression technique finds out a linear relationship between x (input) and y (output).
